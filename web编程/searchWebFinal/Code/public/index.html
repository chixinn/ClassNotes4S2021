民进中央2021年重点考察调研专题座谈会在上海交大召开

4月14日下午，民进中央2021年重点考察调研专题座谈会在上海交通大学闵行校区召开，就“加强科研队伍建设， 为强化国家战略科技力量提供人才保障”进行专题交流。全国政协副主席、民进中央常务副主席刘新成，全国政协副秘书长、民进中央副主席朱永新，全国政协常委、民进中央副主席、民进四川省委会主委、中国科学院成都分院院长张雨东，民进中央副主席、上海市政协副主席、民进上海市委会主委、上海交通大学副校长黄震，全国人大常委会委员、民进中央副主席兼秘书长高友东，上海交大党委书记杨振斌，党委副书记顾锋，副校长毛军发，党委常委、统战部部长张卫刚等出席座谈会。座谈会由黄震主持。杨振斌对各位领导、嘉宾和专家学者来访表示欢迎。他介绍了上海交大“因图强而生、因改革而兴、因人才而盛”的发展历程和“综合性、创新型、国际化”办学新格局。他表示，上海交大将始终坚持以兴邦为任、以育人为本、以创新为魂，落实立德树人根本任务，勇担服务国家时代使命，为国家强盛和民族复兴作出更大贡献。上海交大科学技术发展研究院、人力资源处有关负责同志围绕学校科研创新及科研队伍建设情况，分别作专题发言。数学科学学院李从明，自然科学研究院金石，机械与动力工程学院盛鑫军、徐凯，生物医学工程学院顾力栩等教授先后发言，围绕“支持科研人员潜心研究”“鼓励科技人才发展”“夯实基础研究科研力量”等内容进行了交流探讨。刘新成作总结讲话，他感谢大家的建言献策，表示高校的实践经验、意见建议对本次调研具有重要的参考价值，民进中央调研组将对此次调研获得的材料认真总结，提炼成提案与建议，提交国家有关部门。他希望上海交大立足实践、深入研究，为加强科技管理人才培养，加快推进国家战略科技力量建设贡献更多交大智慧和经验。座谈会前，调研组一行实地考察了大零号湾全球创新创业集聚区，了解总体建设情况，听取上海飒智智能科技有限公司、上海航数智能科技有限公司、上海节卡机器人科技有限公司等企业负责人汇报，与大家面对面交流。参加调研活动的还有，来自科技部、财政部、人力资源和社会保障部、中央统战部相关司局的负责同志，首都师范大学、中国科学院科技战略咨询研究院、湖北省科技厅、中国科学院的相关专家，以及民进中央机关、中共上海市委统战部、民进上海市委会的同志，上海交大有关部处、院系负责人等。 作者： 祁洁 摄影： 梅秋武、王赟喆、华慈恩 供稿单位： 新闻中心
